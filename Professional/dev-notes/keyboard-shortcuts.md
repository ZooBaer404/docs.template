# Keyboard Shortcuts

## App Shortcuts

| Shortcut    | Action              |
|-------------|---------------------|
| `Ctrl + S`  | Save current form   |
| `Shift + N` | New entry dialog    |
| `Alt + ←/→` | Switch between tabs |

---

## Dev Tools (CLI / Emacs)

| Shortcut   | Action                     |
|------------|----------------------------|
| `, r`      | Reload Emacs config        |
| `SPC SPC`  | Search files (like VSCode) |
| `Ctrl + G` | Cancel command             |

> Update this as shortcuts are added or changed.
